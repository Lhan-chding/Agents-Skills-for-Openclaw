# Formula Explanation Protocol

Use this structure for every important formula.

## Template

1. Formula
- Write the exact expression.

2. Symbol table
- Symbol
- Meaning
- Domain or shape
- Unit or scale (if relevant)

3. Pre-conditions
- State assumptions needed for this equation.

4. Derivation
- Use numbered steps.
- One algebraic transform per step.
- Show substitutions explicitly.
- Name applied rules (chain rule, integration by parts, finite difference stencil, etc.).

5. Interpretation
- Explain practical meaning in plain language.
- Provide one concrete mini-example.

6. Validation checks
- Boundary case check.
- Dimensional consistency check.
- Sign/monotonicity sanity check.

## No-Skip Rule

Do not skip intermediate transforms. If the full derivation is too long, split into phases and keep each phase complete.

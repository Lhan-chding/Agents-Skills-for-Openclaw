# Feishu Collaboration Patterns

Use these patterns when assisting live document collaboration.

## Suggestion + Paste Pattern

For each requested edit:

- `修改建议`: one to three bullets
- `可直接粘贴版本`: final paragraph or section

## Terminology Unification Pattern

Provide a compact mapping table:

| Current term | Recommended term | Reason |
| --- | --- | --- |

Apply the mapping consistently in the paste-ready version.

## Structure Cleanup Pattern

When document is messy:

- keep original order unless user requests reorder
- normalize heading levels
- split oversized paragraphs
- preserve references and evidence markers

## Large Rewrite Guard

Before large rewrite, request explicit confirmation and list expected impact:

- changed sections
- removed content
- tone changes
- potential meaning shifts

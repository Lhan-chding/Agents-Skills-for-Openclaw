# Rewrite Strategy

Apply edits in this order:

1. Correctness
- Fix factual or logical errors first.

2. Clarity
- Simplify long sentences.
- Remove ambiguity.

3. Consistency
- Normalize terms, tense, person, and punctuation.

4. Structure
- Improve headings, transitions, and list structure.

5. Tone
- Apply requested tone after content and structure are stable.

## Minimal-Change Default

- Prefer local edits over full rewrites.
- Keep original paragraph boundaries when possible.
- Keep named entities and technical terms stable unless user requests replacement.
